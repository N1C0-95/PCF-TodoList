## Installation
_Per creare la solution da importare:_
 * Visual Studio Code Terminal:
  1) pac solution init --publisher-name YourPub--publisher-prefix YourPfx
  2) pac solution add-reference --path “C:\PCF Controls\RecordGUID” <br>
 * Developer Command Prompt:
  1) MSBUILD /t:restore
  2) MSBUILD
